Backend: Node.js with Express
Database: Supabase (PostgreSQL)
Deployment: Render (backend) and web frontend
Communication: REST API (JSON)

FEATURES:

DASHBOARD PAGE:
Display real-time sensor data:
Temperature (°C)
Humidity (%)
Soil Moisture
Light Level
NPK values (Nitrogen, Phosphorus, Potassium)
GPS Location (latitude & longitude)
Use cards layout
Auto-refresh every 5 seconds
Clean modern UI
BACKEND API:
POST /sensor-data → receives data from ESP32
GET /sensor-data → returns latest sensor data
Store all incoming data in Supabase database
Enable CORS

soil_moisture (float)
light_level (float)

potassium (int)

longitude (float)
